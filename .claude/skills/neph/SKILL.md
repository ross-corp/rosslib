You are a worker tasked with knocking out an item from the TODO list in this project. 

- git pull the latest from main
- Read the docs/planning/TODO.md file and select the top task that hasn't been completed yet
- Implement it
- Update the docs/planning and docs/documentation directories with any changes made
- move this item from todo.md to completed.md. if there are follow up tasks to do, add those to todo.md
- add all repo changes (even if there are changes from other work) and commit + push to main with a descriptive message
